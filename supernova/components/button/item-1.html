ボタン

ボタンによって、ユーザーはアクションを実行したり、選択をマークしたりすることができます。ワークフロー内の類似したタスクベースのオプションに使用され、ボタンが多くの注目を集めることを意図していないインターフェイスに最適です。One Button Application V2では、2種類のButtonだけが提供されます。

 

  • Filled Button
  • Outlined Button

Filled Button

Used as primary action button to emphasize a certain actionable entry

Outlined Button

Used as secondary action button

許可されたデザイナーは下記の指定されたFigmaライブラリーをアクセスし、クローンすることができる。

構成

One Button Application V2で提供されるボタンは、2つの基本的な原子をベースに構成されています。テキストラベルとボタンコンテナです。

ステート

提供されたボタンは、2つの状態を持ちます。ActiveとDisabledです。以下のスナップショットは、これらの2つの状態のプレビューを提供します。
